# 🚀 TkCodes - Static Landing Page

A modern and responsive static landing page built using **HTML5** and **CSS3**. This project demonstrates clean UI design, responsive layouts, smooth navigation, and modern styling techniques.

## 🌐 Live Demo

👉  https://tiyabak.github.io/TkCodes/

## ✨ Features

- 🎨 Modern and clean UI
- 📱 Fully responsive design
- 🧭 Smooth scrolling navigation
- 💻 Hero section with call-to-action buttons
- 📖 About section
- 🛠️ Services section
- 📞 Contact section
- 🦶 Professional footer
- ⚡ Hover animations
- 🌈 Beautiful color palette
